업무 회고록

Posted on Mon, Jul 20, 2020

W30

  • 에어브릿지 Integration 정리 중. 필요없는거 지우고, 설명이 불분명한 것들은 보충하고. 나조차도 잘 모르는 연동(ex. GA)이 있어서, 꼭 직접 해보고 문서 작성을 해둬야 할 듯. 이런 무지는 불편으로 CSM, 고객사에게 나란히 전달되기 마련이다. 정리 마무리하고, 연동 쪽도 Jira 프로젝트 하나 만들어서 TODO/DOING/DONE 리스트 만들고 Integration Squard(Tech Manager, Partner Manager) 하나가 붙어서 KPI(연동 개수 x 연동 고객수) 가지고 하나하나 짜나가면 될 것 같다.
